# Dynamic Notch for Windows

I will give you step-by-step instructions on how to use it and explain what it is.

Dynamic Notch is an Apple-inspired concept. Think of it as a Dynamic Island in a notch form for Windows computers.

## What does it do?

Well, a LOT.You can:

* **View the time and battery indicator**
* **Search directly on it**
* **Control music**
* **Play YouTube videos** in the notch while you do something else
* **Get messages** from your phone onto the notch
* **Get calls and answer them**
* **Use voice search**
* **See widgets** Weather AND Stoinks! more coming soon
* **Display timers**
* **See voice recording indicators**

And the best part? I am constantly adding more stuff to it, making it better, and fixing bugs.

## How to use it?

It is simple: download it from GitHub (the release) , and then run it in the terminal the exact command is npm install first and then npm start.

## How to operate the notch?

Simple. Just hover in, and hover out when you do not want it.

## Operating system?

Windows FOR NOW, because I do not have a Mac to test on. Maybe if I get one from macondo, Haha.

## How to pair messages?

Simple click on the bottom arrow on your computer then it will show apps and then right click on dynamic notch and then click login to messages and then scan a qr code which pairs the messaging system with the notch and you're all set then!
